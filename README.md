# 🌍 CarbonWise

> **Understand, Track, and Reduce Your Carbon Footprint with AI-Powered Personalized Insights**

A modern, mobile-first web application that helps individuals measure their environmental impact, track daily carbon habits, and receive AI-generated personalized suggestions to reduce their footprint.

## 🎯 Problem Statement

> Design a solution that helps individuals understand, track, and reduce their carbon footprint through simple actions and personalized insights.

## 💡 Solution Overview

CarbonWise addresses all three pillars of the problem statement:

| Pillar | Feature | How |
|--------|---------|-----|
| 🧠 **Understand** | Carbon Calculator + Dashboard | Multi-category emission calculation with visual breakdown |
| 📅 **Track** | Daily Habit Tracker | One-click activity logging with 7-day history |
| 🌱 **Reduce** | AI Suggestions Engine | GPT-4o-mini powered personalized tips |

---

## ✨ Key Features

### 1. 🧮 Smart Carbon Calculator
- 6 emission categories: Transport, Electricity, Diet, Shopping, Gas, Flights
- Conditional inputs (LPG days vs PNG bill based on user's gas type)
- Pre-filled with realistic Indian averages for quick testing
- Real-time validation with helpful error messages

### 2. 📊 Insights Dashboard
- Big visual total with color-coded level (Excellent → Critical)
- Category-wise emission breakdown
- Comparison with India, Global, and Sustainable Target averages
- Real-world impact equivalents (trees lost, km driven, phone charges)

### 3. 🤖 AI Personalized Suggestions
- Context-aware tips generated by GPT-4o-mini (via GitHub Models)
- Prioritized by user's highest emission category
- Each suggestion includes estimated CO₂ savings in kg/month
- Tailored for Indian users with culturally relevant advice

### 4. 📅 Daily Habit Tracker
- 19 pre-defined quick-add activities (one-click logging)
- Real-time daily total with color-coded level badges
- 7-day history with trend visualization
- Two-step confirmation for destructive actions (data safety)
- Auto-save to localStorage (no manual save needed)

### 5. 🎨 Beautiful Earth-Themed UI
- Rotating earth orb with CSS animations (no images = fast load)
- Dark cosmic gradient background
- Floating particles for ambient atmosphere
- Custom toast notifications and modals (no ugly browser alerts)
- Fully responsive mobile-first design

## 🛠️ Tech Stack

### Frontend
- **Framework**: Next.js 14 (Pages Router)
- **Language**: JavaScript (ES6+)
- **Styling**: Custom CSS with CSS Variables (no UI library needed)
- **State**: React Hooks (useState, useEffect)
- **Storage**: Browser localStorage (privacy-first)

### Backend (API Routes)
- **Runtime**: Node.js (via Next.js API routes)
- **AI Integration**: OpenAI SDK + GitHub Models (GPT-4o-mini)

### Deployment
- **Hosting**: Vercel
- **CI/CD**: Auto-deploy on GitHub push
- **Environment**: Node 18+

## 🚀 Getting Started

### Prerequisites
- Node.js 18 or higher
- npm or yarn
- A GitHub account (for the AI token)

### Installation

# Clone the repository
git clone https://github.com/YOUR_USERNAME/carbonwise.git
cd carbonwise

# Install dependencies
npm install

# Create environment file
# Copy .env.example to .env.local and add your token
cp .env.example .env.local

Get Your GitHub Token (Free AI Access)
Go to: https://github.com/settings/tokens
Click "Generate new token" → "Classic"
Give it a name (e.g., "CarbonWise AI")
Select scope: models:read
Generate and copy the token
Set Environment Variable
In .env.local:

text

GITHUB_TOKEN=ghp_your_token_here
Run Development Server
Bash

npm run dev
Open http://localhost:3000 in your browser.

🌐 Deployment
Deploy to Vercel (Recommended)
Push your code to GitHub
Go to vercel.com/new
Import your repository
Add environment variable: GITHUB_TOKEN
Click Deploy
The app will be live in ~2 minutes!

🎨 Design Philosophy
Mobile-First
Designed primarily for mobile devices since most users will track habits on their phones.

Privacy-First
All data stays on the user's device. No backend database, no user accounts, no tracking. localStorage handles persistence.

Behavior Change Focus
The app doesn't just show numbers - it provides:

Context (comparisons with averages)
Emotion (real-world equivalents)
Action (specific AI suggestions)
Tracking (daily engagement loop)
Performance
No external UI library (smaller bundle)
CSS animations (no images for icons)
Static optimization where possible
Tree-shaking via named imports

🔐 Security Practices
✅ Environment Variables: API tokens never in code
✅ Input Validation: Client + server-side
✅ Generic Errors: No info leakage in error messages
✅ Method Restrictions: APIs only accept POST
✅ Multi-step Confirmations: For destructive actions
✅ No Third-party Tracking: User privacy respected
